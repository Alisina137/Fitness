You are a senior AI fitness product engineer building a world-class fitness application.

The following systems are already implemented:

* User authentication
* User onboarding and fitness profile
* Exercise intelligence database
* Workout program engine
* Workout tracking and history

Now build the AI Workout Generation Engine.

The goal:

Create an intelligent workout planner that generates personalized workout programs based on each user's body, goals, lifestyle, equipment, experience, and previous performance.

This should become the core AI feature of the application.

Do not create a simple random workout generator.

The AI must use:

* User profile data
* Exercise database
* Workout history
* Performance feedback
* Fitness goals
* Available equipment
* Time availability

---

# 2. Create Workout Generation Rules

The AI must follow these rules.

## Goal-Based Logic

### Fat Loss

Prioritize:

* Full body workouts
* Higher calorie burn
* Strength + conditioning
* Moderate rest periods

### Muscle Gain

Prioritize:

* Progressive overload
* Appropriate volume
* Muscle group frequency
* Hypertrophy rep ranges

### Strength

Prioritize:

* Compound movements
* Lower repetitions
* Higher intensity
* Longer rest

### Endurance

Prioritize:

* Cardio
* Conditioning
* Longer duration

---

Intermediate:

Rules:

* Increased volume

Advanced:

Rules:

# 4. Equipment Intelligence

The AI must never recommend unavailable equipment.

Examples:

User equipment:
"Dumbbells only"

Allowed:

* Dumbbell press
* Goblet squat
* Dumbbell row

Not allowed:

* Barbell squat
* Cable exercises

Create equipment filtering before workout generation.

---

# 5. Injury Safety System

Before generating workouts:

Analyze:

User limitations:

* Knee problems
* Back problems
* Shoulder problems
* Wrist problems

Example:

If knee limitation:

Avoid:

* Heavy squats
* High impact jumping

Suggest:

* Leg press alternative
* Hip exercises
* Mobility work

Safety has higher priority than optimization.

# 7. Create Progressive Overload System

The AI should analyze previous workouts.

Rules:

If user completes all sets easily:

Increase:

* Weight
  or
* Reps
  or
* Difficulty

If user struggles:

Decrease:

* Weight
  or

If user misses workouts:
