You are a senior fitness product engineer building a premium AI fitness application.

The following systems already exist:

* Project architecture
* Authentication
* User onboarding
* Exercise database
* Exercise library

Now build the Workout Program Engine.

The goal:
Create a professional workout management system that can support beginners, advanced athletes, and future AI-generated workout plans.

Do not build AI workout generation yet.

---

# 1. Create Workout Data Architecture

Create scalable database models.

## Workout Plan Model

Fields:

* name
* description
* goal
* difficulty level
* duration in weeks
* created by
* status
* exercises
* schedule
* progression rules

Goals:

* Fat loss
* Muscle gain
* Strength
* Endurance
* General fitness
* Mobility

---

## Workout Day Model

Each workout plan contains multiple days.

Fields:

* day number
* title
* focus area
* estimated duration
* exercises
* completion status

Examples:

Day 1:
Upper Body Strength

Day 2:
Lower Body Strength

Day 3:
Recovery

---

## Workout Exercise Model

Each exercise inside a workout should contain:

* exercise ID
* order
* sets
* repetitions
* weight target
* duration
* rest time
* tempo
* notes

Example:

Exercise:
Barbell Squat

Sets:
4

Reps:
8-10

Rest:
90 seconds

---

## Workout Session Model

Track every completed workout.

Fields:

* user ID
* workout ID
* start time
* end time
* duration
* exercises completed
* calories burned
* notes
* difficulty rating

---
